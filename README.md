# Hong Kong Police Long Service Medal Query System

This project provides a web-platform for searching award-winning policemen via ID number.

> Inspired by an android app called "表揚香港警察"

The data used in this project were retrieved from http://www.gld.gov.hk/ (政府物流服務署) as following:
